# SentinelLLM — AI Data Leakage Detection

Detects PII, secrets, and sensitive data in LLM prompts across enterprise teams.

## Features
- Real-time prompt scanning
- PII & API key detection
- Risk scoring dashboard
- Alert system

## Tech Stack
- FastAPI + Python
- spaCy NER
- React frontend
- SQLite / PostgreSQL
